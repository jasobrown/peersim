Introduction top

Peer-to-peer systems can reach huge dimensions such as millions of nodes, which typically join and leave continously. These properties are very challenging to deal with. Evaluating a new protocol in a real environment, especially in its early stages of development, is not feasible. There are distributed planetary-scale open platforms (e.g., PlanetLab) to develop and deploy network services, but these solutions don't include more than about 440 nodes. Thus, for large-scale systems, a scalable simulation testbed is mandatory.

Peersim has been developed with extreme scalability and support for dynamicity in mind. It is composed of many simple extendable and pluggable components, with a flexible configuration mechanism. To allow for scalability and focus on self-organization properties of large scale systems, some simplifying assumpions have been made, such as ignoring the details of the transport communication protocol stack. Peersim is developed within the BISON project and it is distributed under an open source license. Peersim is written in the Java language.

Additional Code top

We are using Peersim to simulate several P2P protocols. In this section, we make available the code of the more interesting protocols. If the protocol is described in a scientific paper, In this section, we provide both the code of such protocols and the configuration files needed to reproduce the figures of such papers.
  • OverStat is a collection of aggregation protocols. Aggregation is a collective term for a set functions aimed at providing statistical information over a distributed systems, such as the average load incurred by the nodes or the total size of the network. We have developed extremely scalable, robust decentralized aggregation protocols.
    The protocol has been described in the following papers:
    • Alberto Montresor, Márk Jelasity, and Ozalp Babaoglu. Robust Aggregation Protocols for Large-Scale Overlay Networks. In Proceedings of the 2004 International Conference on Dependable Systems and Networks (DSN'04)}, pages 19-28, Florence, Italy, June 2004. IEEE Computer Society. PDF Version
    • Márk Jelasity and Alberto Montresor. Epidemic-Style Proactive Aggregation in Large Overlay Networks. In Proceedings of the 24th International Conference on Distributed Computing Systems (ICDCS'04), pages 102-109, Tokyo, Japan, March 2004. IEEE Computer Society. PDF Version
    A journal version of these papers has been accepted for publication in ACM Theory of Computing Systems (TOCS). The code and the configuration files used to generate the journal version are available here.
  • SG-1 is a protocol for self-organizing and maintaining a superpeer based topology. The protocol has been described in the following paper:
    • Alberto Montresor. A Robust Protocol for Building Superpeer Overlay Topologies. In Proceedings of the 4th International Conference on Peer-to-Peer Computing, Zurich, Switzerland, August 2004. IEEE. PDF Version
    The code and the configuration files used to generate this paper are available here.
