{@link peersim.core.Control} classes for initializing a network or modifying it during simulation.

Among other things, this package includes classes for the following two purposes:

  • wiring: used to wire network topologies like ring lattices, regular random graphs, scale-free networks, etc. It is also possible to wire networks using an arbitrary custom made method using {@link peersim.dynamics.WireByMethod}.
  • churn: used to simulate node addition and removal patterns (aka churn)
